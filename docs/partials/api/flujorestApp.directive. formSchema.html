formSchema
directive in module flujorestApp

Description

Crea un formulario completo al pasarle un schema desde mongodb 

formConfig = {
     formTextContent:{
       mainTitle:"Entrada de Blog",
       submitBtn:"Enviar el formulario"
     },
     name:{
       labelText:"Nombresillo",
       requiredText:"Pon el campo oe",
       order:1
     }
     properties:{
       order:5,
       titleArray:"Propiedades del documento",
       addItemToArray:"Agrega una nueva propiedad",
       deleteArrayItemBtn:"Borra esta propiedad",
       name:{
         labelText:"Nombre dentro de una propiedad",
         requiredText:"No seay choclo",
         order:1
       }
     }
   };

Usage

as element:
<form-schema>
</form-schema>
as attribute
<ANY form-schema>
   ...
</ANY>

Methods

  • schemaToArray(Schema)

    Transforma todos los key de primer nivel y el objeto que tiene la propiedad "instance":"Array" en la ruta schema.paths -- enun array. Ademas, se eliminan todos los metodos que no tienen la propiedad "instance".

    Para limpiar el objeto y dejarlo en JSON: 

                  var cleanSchema = JSON.parse(JSON.stringify(schema));
